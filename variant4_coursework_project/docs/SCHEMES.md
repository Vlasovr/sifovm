# Графическая часть

В каталоге `schemes/` лежат четыре редактируемых листа `.drawio`, а в
`schemes/pdf/` - 16 отдельных PDF-листов формата А3.

По заданию нужен минимум из четырёх листов А3. Для запаса добавлен расширенный
комплект детальных схем по основным узлам.

| Лист | `.drawio` | PDF | Что показывает |
|---|---|---|---|
| А1 | `A1_general_structure.drawio` | `pdf/A1_general_structure.pdf` | общая структурная схема микро-ЭВМ |
| А2 | `A2_control_unit.drawio` | `pdf/A2_control_unit.pdf` | устройство управления, дешифратор, секвенсор фаз и матрица сигналов |
| А3 | `A3_datapath_alu_stack.drawio` | `pdf/A3_datapath_alu_stack.pdf` | РОН, АЛУ, регистр флагов, стек, тракт записи результата |
| А4 | `A4_memory_dma_arbiter.drawio` | `pdf/A4_memory_dma_arbiter.pdf` | ROM/RAM, кэш, DMA/КПДП, арбитр, предсказатель переходов |

## Что нужно сделать перед сдачей

1. Проверить PDF-листы в `schemes/pdf/`.
2. Если преподаватель требует другой штамп, открыть `.drawio` в diagrams.net или draw.io.
3. После правок пересобрать PDF через `tools/export_drawio_to_pdf.py` и `tools/generate_detailed_scheme_pdfs.py`.
4. Вставить экспортированные листы в пояснительную записку и приложения.

## Что обязательно должно быть видно

### А1

- ПЗУ команд;
- IP/IR;
- устройство управления;
- РОН 12 x 16;
- АЛУ `OR/NOR/SRA/INCS`;
- регистр флагов;
- стек 7 x 16 с ростом вниз;
- кэш данных;
- ОЗУ данных;
- предсказатель A4;
- DMA/КПДП;
- арбитр шины.

### А2

- дешифратор команды;
- секвенсор `T0...T5 + Tw`;
- входы `FR.Z`, `FR.S`, `FR.C`, `FR.O`;
- события `CACHE_MISS`, `DMA_WAIT`, `HLT`;
- матрица управляющих сигналов;
- логика загрузки IP.

### А3

- два канала чтения РОН;
- один канал записи РОН;
- MUX входов АЛУ;
- ALU result writeback;
- формирование `Z/S/C/O`;
- `PUSH`/`POP` и изменение `SP`.

### А4

- отдельный ROM-тракт команд;
- RAM-тракт данных;
- `REQ_CPU/GNT_CPU`;
- `REQ_DMA/GNT_DMA`;
- кэш `4-way, 16 sets`;
- замещение по максимальному возрасту строки;
- сквозная запись;
- запись DMA по адресам `000A..000C`;
- предсказатель A4 с `PHT + BTB + GHR`.
