# Техническое задание и состав варианта 4

## Общие требования курсового проекта

Курсовой проект по дисциплине СиФО ЭВМ выполняется на тему «Разработка микро-ЭВМ на ПЛИС».

Обязательный комплект:

- пояснительная записка не менее 30 страниц;
- графическая часть не менее четырех листов А3;
- исходные схемы/файлы проекта Quartus;
- результаты функционального моделирования отдельных блоков и всей системы;
- дампы ROM, RAM и кэш-памяти;
- листинг программы, включающей все команды архитектуры, в символьном и бинарном виде.

## Система команд

Все команды имеют фиксированный двухсловный формат. Первое слово содержит код операции и номер регистра, второе слово содержит прямой адрес операнда или адрес перехода.

| Код | Команда | Действие |
|---|---|---|
| `00h` | `HLT` | останов |
| `01h` | `MOV adr, reg` | `M[adr] -> Rn` |
| `02h` | `MOV reg, adr` | `Rn -> M[adr]` |
| `03h` | `OR reg, adr` | `Rn := Rn OR M[adr]` |
| `04h` | `NOR reg, adr` | `Rn := NOT(Rn OR M[adr])` |
| `05h` | `SRA reg` | арифметический сдвиг вправо |
| `06h` | `INCS reg` | `Rn := Rn + FR.S` |
| `07h` | `PUSH reg` | помещение регистра в стек |
| `08h` | `POP reg` | извлечение из стека в регистр |
| `09h` | `JMP adr` | безусловный переход |
| `0Ah` | `JZ adr` | условный переход по `Z = 1` |

Команда `JZ` добавлена для демонстрации работы предсказателя переходов.
