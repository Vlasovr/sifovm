# Листинг тестовой программы

Формат команды фиксированный, два слова по 16 бит:

- `word0[15:8]` - код операции;
- `word0[7:4]` - номер регистра `R0..R11`;
- `word0[3:0]` - резерв;
- `word1[15:0]` - адрес операнда или адрес перехода.

## Символьный и шестнадцатеричный вид

| Адрес ROM | Word0 | Word1 | Команда | Назначение |
|---:|---:|---:|---|---|
| `0000h` | `0110h` | `0020h` | `MOV 0020h, R1` | загрузка `8001h` |
| `0002h` | `0510h` | `0000h` | `SRA R1` | `8001h -> C000h`, `C=1` |
| `0004h` | `0610h` | `0000h` | `INCS R1` | `C000h + S -> C001h` |
| `0006h` | `0710h` | `0000h` | `PUSH R1` | запись `C001h` в стек |
| `0008h` | `0120h` | `0021h` | `MOV 0021h, R2` | загрузка `00F0h` |
| `000Ah` | `0320h` | `0022h` | `OR R2, 0022h` | `00F0h OR 0F0Fh = 0FFFh` |
| `000Ch` | `0420h` | `0023h` | `NOR R2, 0023h` | `NOT(0FFFh OR 0000h) = F000h` |
| `000Eh` | `0830h` | `0000h` | `POP R3` | извлечение `C001h` |
| `0010h` | `0230h` | `0030h` | `MOV R3, 0030h` | запись `C001h` в RAM |
| `0012h` | `0140h` | `0025h` | `MOV 0025h, R4` | загрузка `FFFFh` |
| `0014h` | `0440h` | `0025h` | `NOR R4, 0025h` | получение `0000h`, `Z=1` |
| `0016h` | `0A00h` | `001Ah` | `JZ 001Ah` | переход должен выполниться |
| `0018h` | `0150h` | `0024h` | `MOV 0024h, R5` | должна быть пропущена |
| `001Ah` | `0900h` | `001Eh` | `JMP 001Eh` | безусловный переход |
| `001Ch` | `0160h` | `0024h` | `MOV 0024h, R6` | должна быть пропущена |
| `001Eh` | `0170h` | `000Ah` | `MOV 000Ah, R7` | чтение первого слова DMA |
| `0020h` | `0000h` | `0000h` | `HLT` | останов |
